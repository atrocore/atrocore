AtroCore provides integration capabilities that allow you to connect with external services and cloud storage platforms. These integrations enable seamless data exchange and file management across different systems.

The integration modules support various external services including [Microsoft 365](https://store.atrocore.com/en/microsoft-365-connector/20205) services and cloud storage solutions. These integrations provide automatic synchronization capabilities, allowing you to manage files stored in external systems directly from within AtroCore while maintaining data consistency across platforms.
