The [Reports](https://store.atrocore.com/en/reports/20213) module lets you create tabular summaries of any entity's data. Records are grouped by one or more fields, and aggregation functions calculate statistics across each group.

Completed reports can be pinned to the [Dashboard](../../01.atrocore/07.dashboards) as dashlets.
